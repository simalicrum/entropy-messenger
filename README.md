# entropy-messenger

Entropy Messenger is a simple Web based messenger client similar to Telegram. Entropy was built with [React](https://github.com/facebook/create-react-app). It uses [Semantic UI React](https://react.semantic-ui.com/) for css and [Google Firebase](https://firebase.google.com/) for the backend.

Live server: https://simalicrum.github.io/entropy-messenger/

This is my [Javascript Final Project](https://www.theodinproject.com/courses/javascript/lessons/final-project), a component of [The Odin Project](https://www.theodinproject.com/) full stack web development course.
